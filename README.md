# Tilda mark blocks

Плагин читает файл https://dc.ria.ru/ips/t/lib/plugin/list.json

## Scripts

```json
// install dependencies
npm install

// build extension and watch for changes
npm run dev

// build extension zip
npm run build

// lint all source files
npm run lint
```
